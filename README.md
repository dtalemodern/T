# Work Report App

This is a simple Python command-line application that asks you two questions
about your work and generates a short report.

## Usage

Run the script using Python 3:

```bash
python3 work_report.py
```

The script will ask:
1. What tasks did you complete today?
2. What challenges did you face?

After you answer the questions, it prints a formatted report and
saves it to `report.txt` in the current directory.
