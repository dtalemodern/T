# Work Visual App

A simple mobile-style web application to manage tasks visually. It's built with plain HTML, CSS and JavaScript.

## Usage

Open `index.html` in your browser. You can add tasks using the input field, mark them as done or undo them. Tasks are stored in your browser's local storage.

This is just a basic example to demonstrate a visual approach compared to a pure text-based tool. For a more full-featured mobile app, consider using frameworks like React Native or Flutter.
